# Computer Communication Networks
### Experiments based on different concepts of CCN carried out using Juniper EX2300-C running on JUNOS-OS  
### List of experiments:
- Cyclic Redundancy Check
- Chat (Client - Server)
- Echo (Client - Server)
- FTP (Client - Server)
- Single Switch VLAN
- Two Switch VLAN
- VLAN with Trunk Link
- DHCP
- OSPF
- IBGP
